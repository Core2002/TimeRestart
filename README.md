<h1 align="center">🪁 Time Restart 择机重启</h1>
<h5 align="center">当在线人数小于等于某阈值时尝试重启以应用服务端更改
</h5>

------

## 介绍
在管理服务器时，需要对服务端重启，  
然后玩家又很多，贸然重启会影响玩家游戏体验，甚至导致玩家流失。  
因此，我们需要等玩家人数少的时候再进行重启。  
但是等待的过程太漫长了，一直守着也不是个事。  
于是就有了此项目。  


## 例子
比如输入命令 `/ts 3`  
然后当在线人数还剩 `3` 个人的时候，就重启  


> 注意事项：  
> 请自行编写 `启动脚本` 实现服务自启
